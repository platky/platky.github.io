Address Order


Information

Utilizing Google Maps api Address Order finds and displays the optimal path between a multi-waypoint route. My brother needed a simple system for delivering several pizzas in a row so I threw this together. It can be utilized fully at the link on the right

Tools

Outside of the typical web application languages I utilized google maps for both displaying and calculating routes.
